# UX Rules

## Regra principal

O aluno nao deve sentir que esta falando com uma caixa-preta magica.

Ele deve entender:

```text
Hoje vou fazer isso.
Porque isso melhora meu proximo contato com ingles.
```

## Nao mostrar engenharia

Evite no fluxo diario:

- Snowball;
- flywheel;
- automaticity debt;
- valor composto;
- score tecnico.

Use linguagem humana:

- proximo passo;
- expressao valiosa;
- menos legenda;
- vida real;
- progresso.

## Comandos humanos

```text
começar
continuar
energia baixa
ver progresso
```

## Templates obrigatorios

Missao diaria:

```text
Missao de hoje
Tempo:
Objetivo:

Hoje voce vai fazer:

Por que:

1.
2.
3.

Primeira acao:
```

Modo retorno:

```text
Voce nao perdeu nada.
Vamos recuperar em 5 minutos.
```

Modo guerra:

```text
Grave ou fale 30 segundos.
Acabou.
```

# Instalação e atualização da V1 Snowball

1. Faça backup de `.ingles-em-contexto/`.
2. Instale o agente V1.
3. Copie os novos tools.
4. Preserve `EVENTS.jsonl`, `MASTERY.json` e `STATE.md`.
5. Adicione os arquivos da V1 ausentes.
6. Execute:

```text
validar memória
validar motor
inicializar snowball
```

A ferramenta de memória usa schema interno próprio.

O Snowball usa schema interno próprio para:

- `SNOWBALL_CAPITAL.json`;
- `TRANSFER_GRAPH.json`;
- `SERIES_CORPUS.json`;
- `INCIDENTAL_CANDIDATES.json`;
- `AUTOMATICITY_DEBT.json`.

Os dados antigos não devem ser apagados.
